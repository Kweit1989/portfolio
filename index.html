Мои проекты

Вот мои проекты в области front-end разработки, выполненные с максимально точным соблюдением принципа Pixel Perfect. Каждый проект обладает адаптивным дизайном, что обеспечивает отличное отображение на любых устройствах.

Elegant

На странице Elegant интерактивность была добавлена с помощью JavaScript, включая реализацию выпадающего меню при наведении курсора. Слайдер для просмотра товаров интегрирован с использованием сторонней библиотеки Swiper. Также с помощью JavaScript был реализован таймер, отсчитывающий время до окончания действия скидки на товар от текущего момента. Проект был создан с помощью Gulp

Loft-hous

Для улучшения функциональности сайта Loft-hous добавлены элементы на JavaScript, включая интерактивную Yandex карту с указанием местоположения компании. В форму обратной связи интегрирована проверка номера телефона на соответствие международному формату. Также реализовано бургер-меню с выезжающим списком для мобильных устройств, а видео с YouTube встроено с учетом корректного отображения на странице.
